**更新日期**：2026-03-17

## 一、海外主要竞品

### Stylebook
- **定价**：$4.99 买断，iOS 独占
- **上线时间**：2009 年，老牌产品
- **核心亮点**：90+ 功能，以数据统计见长
- **优势**：
  - Cost-per-wear 计算器，追踪每件单品价值
  - 穿搭日历 + 出行打包清单
  - 丰富统计（最常穿/最少穿/最贵/颜色分布等）
  - AI 自动抠图，iCloud 多设备同步
  - 数据完全私有，不对外售卖
- **劣势**：AI 搭配建议弱、界面老旧、仅限 iOS

### Cladwell
- **定价**：免费（限每日 1 套）/ $7.99/月（无限功能）/ $49/月（真人造型师）
- **核心亮点**：胶囊衣橱理念 + ChatGPT 接入
- **优势**：
  - 预设 30+ 胶囊衣橱模板，快速上手
  - "Ask Cladwell" 对话式搭配问答
  - 天气感知的每日穿搭推荐
- **劣势**：统计/cost-per-wear 需付费；强推预设胶囊而非用户真实衣橱；付费墙高

### Whering
- **定价**：免费（主要功能）
- **核心亮点**：可持续时尚 + 社区驱动
- **优势**：
  - 统计、cost-per-wear 对免费用户开放
  - Chrome 插件一键导入电商单品
  - "Dress Me" 随机穿搭功能
  - 环境影响追踪（碳足迹统计）
  - 社区分享与互动
- **劣势**：AI 搭配能力一般，虚拟试穿缺失

### Acloset
- **定价**：免费（≤100 件）/ 付费 Pro
- **核心亮点**：AI 能力最强的视觉型衣橱
- **优势**：
  - Smart Detector 自动识别 + 抠图
  - AI 穿搭建议、虚拟试穿
  - 天气预报联动
  - Cost-per-wear、多维统计
  - 多衣橱管理
- **劣势**：免费版 100 件上限限制明显；社区互动浅（只能看，不能互动）；虚拟试穿效果粗糙（2D 拼图式）

### TryonMuse
- **定价**：订阅制
- **核心亮点**：AI 原生，用户留存率行业领先
- **数据**：日活参与度是行业均值的 4.2 倍，留存率 68% vs 行业 42%
- **劣势**：产品较新，生态和社区体量小；缺乏衣橱管理和统计功能

## 二、国内主要竞品

### 搭搭
- **定价**：免费增值
- **用户规模**：200 万用户（2024 年数据）
- **优势**：
  - AI 深度学习，学习千万网红/明星穿搭
  - 批量上传 + AI 自动抠图 + 自动识别分类
  - 天气温度智能推荐 OOTD
  - 电商导入（淘宝等）
  - 多维统计 + 穿衣成本记录
  - 家庭衣橱、云同步多端
- **劣势**：社区功能相对弱；无虚拟试穿

### 穿搭助手
- **用户规模**：800 万用户
- **优势**：
  - AI 多维测试：体型 / 肤色 / 脸型 / 风格 / 发型
  - 虚拟穿搭展示
  - 个性化商城 + 一键跳转淘宝
- **劣势**：重导购变现，轻衣橱管理工具；虚拟试穿效果一般

### 今日装（OOTD）
- **优势**：
  - 单品属性最全（价格/品牌/季节/色系/风格/场合）
  - 场合筛选（约会、工作、运动等）
  - 衣橱统计（穿着频率、性价比、单次费用）
  - 淘宝一键导入
- **劣势**：社交/分享功能弱；操作路径较繁琐；无 AI 搭配和虚拟试穿

### 衣橱大爆炸
- **优势**：
  - 社交属性强："Ta 的衣橱"可查看他人单品与搭配
  - 默认关注搭配达人，灵感获取便捷
  - 总览页体验好，操作简洁
- **劣势**：无衣橱统计；不支持场合标注；无 AI 能力

## 四、OOTD 差异化优势分析

### 4.1 独有能力（竞品均无）

| 能力 | 说明 | 竞品现状 |
|------|------|---------|
| **AIGC 虚拟试穿** | 扩散模型生成用户穿着效果图，效果接近真实照片 | Acloset 有 2D 拼图式试穿，效果粗糙；其余竞品均无 |
| **AI 五维评分** | 多模态 LLM 对效果图进行色彩/风格/时尚/实穿/创意五维打分 | 全行业无此功能 |
| **目标日期智能选择** | 中午前推荐今天穿搭，中午后自动切换为明天，支持 3 天预报 | 竞品天气联动均为当日实时天气 |
| **LLM Pipeline 推荐** | 智能匹配 → 虚拟试穿 → AI 评分 → 排序，端到端 AI 驱动 | 竞品搭配建议多为规则匹配或简单 AI，无试穿+评分闭环 |

### 4.2 领先能力（做得比竞品好）

| 能力 | OOTD | 最强竞品 |
|------|------|---------|
| **AI 搭配推荐** | LLM（Qwen-Max）理解颜色/风格/天气语义 | Cladwell ChatGPT（通用对话，不理解衣橱数据结构） |
| **天气联动深度** | 3 天预报 + 穿衣指数 + 目标日期 + LLM prompt 注入 | 搭搭/Acloset（仅当日温度参考） |
| **AI 识别 + 抠图** | 上传即自动抠图 + 识别分类/颜色/风格/季节/场合 | 搭搭（类似）、Acloset（仅识别+抠图，属性较少） |

### 4.3 需要追赶的能力

| 能力 | 差距 | 规划 |
|------|------|------|
| **电商导入** | 搭搭/今日装已有淘宝导入，Whering 有 Chrome 插件 | Phase 2 实现 |
| **衣橱统计** | Stylebook/Whering/搭搭统计功能丰富 | Phase 2 + LLM 诊断差异化 |
| **穿搭日历** | 几乎所有竞品都有 | Phase 1（OOTD 记录） |
| **社区分享** | Whering/衣橱大爆炸社交功能强 | Phase 3 |
| **多端同步** | 竞品均支持多端，OOTD 当前仅 Web | Phase 3（iOS） |

## 五、竞争策略总结

```
                    AI 能力强
                      ↑
                      |
          Acloset     |    OOTD ★
          搭搭        |    TryonMuse
                      |
 工具导向 ←───────────┼───────────→ 体验导向
                      |
          Stylebook   |    衣橱大爆炸
          今日装       |    Whering
                      |
                      ↓
                    AI 能力弱
```

**OOTD 的定位**：右上角 —— AI 能力最强 + 体验导向。

**核心打法**：
1. **短期壁垒**：AIGC 试穿 + AI 评分，全行业独有，体验碾压
2. **中期护城河**：对话式 AI 顾问（基于真实衣橱），比 Cladwell 的通用 ChatGPT 更精准
3. **长期飞轮**：用户穿着数据积累 → LLM 诊断更准 → 推荐更好 → 用户更活跃 → 数据更多

**不做的事**：
- 不做导购变现（与穿搭助手区隔）
- 不做胶囊衣橱模板（与 Cladwell 区隔）
- 不做碳足迹追踪（中国市场 ROI 低）

## 六、市场机会观察

**竞品普遍存在的空白：**
- AI 搭配 + 虚拟试穿 + 衣橱统计 三合一体验，目前无产品做到
- 国内产品普遍重导购变现，轻工具体验（用户感知"被卖东西"）
- 天气联动停留在"当日温度参考"，无产品做到多日预报 + 目标日期智能选择
- 虚拟试穿体验普遍粗糙，仅 Acloset 有但为 2D 拼图效果
- 无产品提供 AI 穿搭评分能力
- 海外产品缺乏中国电商（淘宝/京东/小红书）的生态整合

**市场数据：**
- outfit planner app 市场 2020-2025 增长 284%
- 2025 年市场估值约 23 亿美元
- AI 原生产品（如 TryonMuse）用户留存率达 68%，远超行业均值 42%
