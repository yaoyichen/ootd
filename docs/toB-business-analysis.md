## 三、核心问题：为什么卖家不自己调模型？

公开可用的模型（Qwen-Image、Kolors、SDXL、IDM-VTON、CatVTON 等）确实能实现虚拟试穿。但实际上绝大多数卖家不会自己做：

### 3.1 卖家不会自己做的原因

| 障碍 | 说明 |
|------|------|
| 技术门槛 | 电商卖家是运营人员，不是开发者。调 API、写代码、部署服务对他们来说不现实 |
| 效果调优 | 好效果不是模型本身决定的，而是 prompt 工程、negative prompt、分辨率参数、图片预处理流程的综合结果。这些 know-how 是核心壁垒 |
| 时间成本 | 卖家的时间花在选品、运营、客服上，花 ¥1-2 买一张图远比自己折腾划算 |
| 一致性 | 产品化工具能保证批量出图质量稳定，自己调 API 效果参差不齐 |
| 基础设施 | 需要 GPU 服务器、图片存储、任务队列等基础设施，个人搭建成本高 |

### 3.2 真正的竞争对手

**不是开源模型，而是 WeShop 这类 SaaS 产品。** 竞争的本质是产品化能力和场景理解，不是底层技术。

## 四、我们的差异化优势

### 4.1 核心差异："搭配推荐 + 试穿"一体化

| 能力 | 竞品（WeShop 等） | 我们 |
|------|-------------------|------|
| 单件试穿 | ✅ | ✅ |
| 多件搭配试穿 | ❌ | ✅ 上衣+下装组合试穿 |
| AI 搭配推荐 | ❌ | ✅ LLM 智能匹配 |
| AI 五维评分 | ❌ | ✅ 色彩/风格/时尚度/实穿性/创意度 |
| 搭配评语 | ❌ | ✅ AI 生成专业评语 |
| 天气适配推荐 | ❌ | ✅ 根据天气推荐搭配 |

### 4.2 对卖家的价值

- **不只帮你拍照，还帮你想怎么搭配卖** — 对卖套装、做穿搭内容的店铺价值巨大
- **一次上传，多套搭配** — 一件上衣 × 多条裤子 = 批量生成搭配效果图
- **AI 评分筛选** — 自动筛出最佳搭配，减少人工判断成本
- **内容素材** — 搭配推荐 + 评语可直接用于商品详情页、短视频脚本

### 4.3 技术壁垒

| 壁垒层 | 说明 |
|--------|------|
| Prompt 工程 | 匹配 prompt、评分 prompt、试穿 prompt 经过大量调优 |
| Pipeline 编排 | 天气→匹配→试穿→评分 四步流水线，SSE 实时推送 |
| 缓存体系 | 同一搭配组合不重复生成，天气按小时缓存，大幅降低成本 |
| 评分体系 | 五维雷达图 + 毒舌评语，竞品没有的差异化体验 |

## 五、目标客户

### 5.1 客户分层

| 层级 | 客户画像 | 需求特点 | 预估数量 |
|------|---------|---------|---------|
| 第一优先 | 跨境电商服装卖家（SHEIN 供应商、亚马逊、Temu） | 需要多种族模特，不绑定国内平台，付费意愿强 | 数十万家 |
| 第二优先 | 1688/淘宝中小服装卖家（白板图卖家） | 有大量白板商品图，急需模特图提升转化率 | 数百万家 |
| 第三优先 | 服装品牌/设计师 | 新品上市前快速生成搭配 lookbook | 数万家 |
| 第四优先 | 穿搭博主/MCN | 批量生成穿搭内容素材 | 数万人 |
| 第五优先（ToC） | 爱穿搭的年轻女性用户 | 享受"虚拟换装"乐趣，想看不同搭配上身效果，有消费力 | 数千万人 |

### 5.2 为什么跨境卖家是第一优先

1. 不在阿里/字节生态内，无法使用免费工具
2. 需要白人、黑人、东南亚等多种族模特，传统拍摄成本极高（$200-500/look）
3. SKU 多、上新快，对批量出图需求强烈
4. 付费习惯好，习惯为 SaaS 工具付费
5. 搭配推荐功能对"套装销售"场景价值明显

### 5.3 ToC 用户群体分析（第二曲线）

#### 用户画像

- 喜欢网购但纠结"穿在我身上什么效果"的年轻女性（18-35 岁）
- 穿搭爱好者，想看"这件上衣配那条裙子"的上身效果
- 有消费力但不一定每件都买，享受"虚拟换装"的乐趣
- 社交分享欲强，愿意把试穿效果图发朋友圈/小红书

#### 触达方式

| 渠道 | 方式 | 预期效果 |
|------|------|---------|
| 小红书 | 发"AI 帮你试穿 100 件衣服"等内容，展示虚拟换装效果 | 天然吸引穿搭爱好者，容易种草传播 |
| 抖音短视频 | "上传你的照片，AI 秒换 10 套 look"视觉冲击力内容 | 容易爆款，获取大量自然流量 |
| 微信小程序 | 做"AI 试衣间"小程序，上传自拍即可试穿 | 社交裂变传播，分享即获客 |

#### 付费模式

ToC 用户不适合 ToB 的月度订阅模式，更适合：

| 模式 | 定价 | 说明 |
|------|------|------|
| 免费试用 + 付费解锁 | 免费 3 次，之后 ¥1/次 | 低门槛，按需付费 |
| 轻量会员 | ¥9.9/月（30 次试穿） | 高频用户订阅 |
| 电商导购分佣 | 免费使用，试穿后跳转购买链接 | 靠 CPS 佣金变现 |
| 广告变现 | 免费使用，展示品牌广告 | 流量规模起来后的变现方式 |

#### 与 ToB 的优先级关系

> **建议当前阶段先聚焦 ToB 卖家，ToC 作为第二曲线。** 原因：
> 1. ToB 客单价高（¥99-999/月），ToC 客单价低（¥1-10/次），早期需要高客单价验证商业模式
> 2. ToB 需求刚性（卖货必须有模特图），ToC 需求弹性（娱乐性质，留存待验证）
> 3. ToB 跑通后积累的试穿能力可直接复用到 ToC，边际成本极低
>
> **启动时机**：ToB 达到 100+ 付费客户后，可在小红书/抖音发内容试水 ToC，根据自然流量和互动数据决定投入力度。

## 十一、总结

### 核心判断

1. **市场真实存在** — 数百万服装电商卖家有白板图转模特图的刚需，愿意为此付费
2. **技术不是壁垒，产品化才是** — 竞争的本质是谁能把 AI 能力包装成卖家最容易用的工具
3. **差异化机会明确** — "搭配推荐 + 试穿"一体化是竞品都没做的方向
4. **跨境电商是最佳切入点** — 不受大厂免费工具挤压，付费意愿强，需求明确
5. **变现路径清晰** — 按张计费模式简单直接，毛利率 75-85%，找到 10 个付费客户即可验证
