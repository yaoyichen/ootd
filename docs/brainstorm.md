# 每日穿搭应用 - 脑暴

**更新日期**：2026-03-17（对齐 MVP 开发进展）

---

## 核心功能方向

### 1. 衣橱管理 ✅ 已实现
- 拍照/上传/拖拽/粘贴单品，AI 自动抠图去背景
- AI 自动识别：类别、颜色、风格、季节、场合
- 标签系统：用户可手动修改所有 AI 识别字段
- 支持录入品牌、价格、购入日期、备注

### 2. 穿搭生成 ✅ 已实现
- LLM（Qwen-Max）智能匹配搭配组合，理解颜色/风格/天气语义
- AIGC 虚拟试穿（qwen-image-2.0-pro），生成用户真实穿着效果图
- 多模态 AI 五维评分（色彩和谐/风格统一/时尚度/实穿性/创意度）
- 天气联动：3 天预报 + 穿衣指数 + 目标日期智能选择
- 同一搭配组合缓存复用，不重复生成

### 3. OOTD 记录 🔜 Phase 1
- 每日打卡，拍照记录当天穿搭
- 日历视图回顾历史穿搭
- 避免短期内重复穿搭
- 自动关联已穿着单品，更新穿着频次

### 4. AI 穿搭顾问 🔜 Phase 1
- 对话式问答，基于用户**真实衣橱**回答（非通用建议）
- "这条白裙子配什么鞋？" → 从衣橱推荐具体单品
- "下周去三亚出差，帮我规划穿搭" → 查目的地天气 + 逐日规划
- "我衣橱里缺什么？" → 衣橱缺口分析

### 5. 快速穿搭 🔜 Phase 1
- "随便穿穿"一键推荐，从历史高分搭配秒出结果
- 不触发完整 pipeline，零 API 调用
- 卡片翻转动画，不满意可"换一套"

### 6. 社区 & 灵感 🔜 Phase 3
- 发现页：浏览他人穿搭
- 按场合/风格/体型筛选参考
- 收藏灵感板（类似 Pinterest）

## 核心产品愿景

### 产品主链路（已跑通）

```
1. 拍照/上传单品
       ↓ AI 自动抠图 + 识别分类/颜色/风格/季节/场合
2. 上传个人照片（全身人像）
       ↓ 支持多人像管理与切换
3. 每日打开 App
       ↓ 自动识别「为今天准备」or「为明天准备」
       ↓ 读取目标日期天气预报 + 穿衣建议
       ↓ LLM 智能匹配 5 组搭配
       ↓ AIGC 批量生成虚拟试穿效果图
       ↓ AI 五维评分 → Top 3 排序展示
4. 用户选择一套 → 出门
       ↓ 可选：拍照打卡 OOTD
```

### 与竞品的核心差异

现有竞品（Acloset、搭搭、Cladwell 等）只能给出「搭配清单」或粗糙的 2D 拼图，
**本产品直接生成用户穿上自己衣服的高质量效果图** — 所见即所得。

加上 AI 五维评分 + 目标日期天气联动 + LLM 端到端 pipeline，形成全行业独有的完整体验闭环。

### 技术实现（MVP 已落地）

| 组件 | 方案 | 说明 |
|------|------|------|
| **虚拟试穿** | DashScope qwen-image-2.0-pro | 768×1152 分辨率，上半身效果好 |
| **单品抠图** | DashScope qwen-image-edit | 去除背景 + 人物模特 |
| **AI 识别** | Qwen-VL-Max（多模态） | 自动识别类别/颜色/风格/季节/场合 |
| **智能匹配** | Qwen-Max（LLM） | 理解颜色语义、风格协调、天气适宜 |
| **穿搭评分** | Qwen-VL-Max（多模态） | 五维独立评分 + 搭配评语 |
| **天气服务** | 和风天气 API | 3 天预报 + 穿衣指数，按小时缓存到 SQLite |
| **城市搜索** | 本地 900+ 城市 JSON | 前端即时过滤，零 API 调用 |

### 主要技术挑战

| 挑战 | 状态 | 说明 |
|------|------|------|
| 试穿一致性 | ⚠️ 基本解决 | 同一人像+衣物组合缓存复用 |
| 多件叠穿 | 🔜 Phase 2 | 上衣+裤子+外套+鞋同时上身 |
| 生成速度 | ⚠️ 优化中 | 单次 ~15s，天气数据已缓存至毫秒级 |
| 细节还原 | ⚠️ 持续改进 | logo/面料纹理/颜色准确性依赖模型升级 |
| 冷启动 | 🔜 Phase 2 | 电商导入（淘宝/小红书）解决上传门槛 |

## 产品定位（已确定）

> **AI 造型师型** — 深度分析衣橱，提供专业搭配建议，生成真实穿着效果图

~~1. 极简工具型 — 快速记录 + 天气推荐，轻量高频使用~~
~~2. 社区分享型 — 以 OOTD 打卡和灵感发现为核心~~
**3. AI 造型师型 — 深度分析衣橱，提供专业搭配建议** ← 选定
~~4. 胶囊衣橱型 — 帮用户建立更有品质感的精简衣橱~~

选择理由：
- AIGC 虚拟试穿是全行业独有壁垒，必须围绕它构建体验
- AI 造型师定位与"所见即所得"核心价值主张最吻合
- 社区/工具属性作为补充，不作为核心（Phase 3 再做社区）
- 订阅制变现，不做导购（与穿搭助手等竞品区隔）
